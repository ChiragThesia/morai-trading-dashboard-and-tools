# Grouped Data

Topic-organized markdown files created by categorizing content from `raw/`. This is the primary reference layer for trading agents.

## Organization

Files are grouped by trading domain:

### Options Strategies
- `iron_condor.md`, `butterfly.md`, `calendar_spread.md`, `vertical_spread.md`
- `covered_call.md`, `wheel_strategy.md`, `straddle.md`, `strangle.md`
- `earnings_plays.md`, `straddle_strangle_strategies.md`

### Greeks
- `greeks_delta.md`, `greeks_gamma.md`, `greeks_theta.md`, `greeks_vega.md`
- `options_pricing.md`

### Market Analysis
- `market_breadth.md`, `market_momentum.md`, `market_patterns.md`, `market_statistics.md`
- `macro_equity.md`, `macro_rates.md`

### Trading Psychology
- `psychology_emotions.md`, `psychology_focus.md`, `psychology_performance.md`

### Quantitative
- `quant_factors.md`, `quant_ml.md`, `backtesting.md`
- `quantocracy_*` files — research by sub-topic

### Volatility
- `volatility_iv.md`, `volatility_skew.md`

### Source-Specific Groupings
- `steadyoptions_*.md` — strategies grouped from SteadyOptions
- `traderfeed_*.md` — topics grouped from TraderFeed
- `abnormalreturns_*.md` — commentary grouped by publication
- `quantocracy_*.md` — research grouped by methodology

### Meta / Reports
- `*_REPORT*`, `*_SUMMARY*` — categorization process documentation
- `PROJECT_COMPLETION_REPORT.md` — overall grouping project status

## Conventions

- Files use underscores for multi-word topics (`greeks_delta.md`)
- Source-prefixed files (`steadyoptions_*.md`) group content from a single source by sub-topic
- Non-prefixed files (`iron_condor.md`) aggregate content across all sources for that topic
- Report files document the categorization methodology and are not trading content
