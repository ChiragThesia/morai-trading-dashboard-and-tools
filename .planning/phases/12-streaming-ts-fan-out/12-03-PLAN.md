<objective>
Expose the sidecar's stream to the TS server and wire the streamer into the app lifecycle.
Three new FastAPI routes — GET /sidecar/events (an internal SSE endpoint draining the streamer's
event queue), GET /sidecar/positions (the cold-start/reconnect reconcile source, STRM-05), and
POST /sidecar/subscribe (the ad-hoc activation endpoint that drives a live subscription for any
OCC symbol, D-05/SC6) — plus the main.py changes that include the routers and launch the streamer
background task at the correct point in the advisory-lock lifecycle.

Purpose: STRM-05 reconcile endpoint + the sidecar→server SSE transport that the server fan-out
(12-05) consumes; the streamer (12-02) is started exactly once, after the lock; and the ad-hoc
activation hook that closes the SC6 end-to-end path (browser → server → sidecar → live ticks).
Output: stream_proxy.py (events + subscribe), positions_proxy.py, streamer.py activation
coordinator, main.py wiring + pytest suites.
</objective>

| server → sidecar (internal SSE/REST) | TS server reads /sidecar/events + /sidecar/positions and POSTs /sidecar/subscribe over the Railway private net |
| trader API → sidecar | Positions data fetched with the trader token (kept fresh by Phase 11 keep-alive) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-03-01 | Information Disclosure | /sidecar/events + /sidecar/positions + /sidecar/subscribe exposure | mitigate | Internal Railway private-net only (GW-05) — no public domain; documented in module docstrings |
| T-12-03-02 | Information Disclosure | error logging on the reconcile path | mitigate | Log type(exc).__name__ only; never str(exc), token, or account values (token_store V6) |
| T-12-03-03 | Tampering | malformed timestamps to the TS Zod boundary | mitigate | utc_now_z on asOf and every emitted ts (Pitfall 5) so Zod .datetime() accepts the payload |
| T-12-03-04 | Elevation of Privilege | streamer launched before lock | mitigate | start_streamer task created only in the post-lock init path (login-after-lock, Pattern 1) |
| T-12-03-05 | Tampering | POST /sidecar/subscribe symbol input | mitigate | Pydantic OCC-format validator rejects malformed symbols (422) before they reach the live StreamClient; the SubscriptionManager 490-cap LRU (12-02) backstops unbounded growth, and authorization is enforced one hop up at the JWT-gated server route (12-05) |
| T-12-SC | Tampering | pip installs | accept | Zero new PyPI packages — StreamingResponse + existing FastAPI only (RESEARCH audit) |
</threat_model>
