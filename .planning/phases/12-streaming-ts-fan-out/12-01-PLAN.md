<objective>
Establish the typed contracts and pure domain math that the entire Phase 12 streaming
pipeline is built on: the sidecar→server→browser SSE payload schemas, the BSM live-greek
recompute (D-02), the `ForReconcilingPositions` driven port + in-memory twin, and the
architecture decision record for the new fan-out pattern (docs-before-code, workflow rule).

parses against these contracts and calls this recompute. Writing it first lets Waves 2-3
proceed in parallel against a frozen shape.

Purpose: One Zod source for stream payloads (MCP-02 single-source discipline); one tested
BSM recompute so live numbers match the journal's math (D-02); a documented architecture
seam before any framework code lands.
Output: stream-events contract, core streaming port + recompute, memory twin, architecture docs.
</objective>
