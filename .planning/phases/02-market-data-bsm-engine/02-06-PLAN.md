<objective>
The compute half of Phase 2: a batch use-case that drains pending observations
(`bsm_iv IS NULL AND mark IS NOT NULL`), inverts IV per row using the stored FRED rate
(4.5% fallback), computes greeks, and writes the five `bsm_*` columns alongside the
vendor-raw values. Mathematically unsolvable rows are stamped `bsm_iv='NaN'` so they fall
out of the pending scan and stay queryable (D-09). Implements BSM-03.

Purpose: This converts stored raw quotes into the trusted own-solver greeks that are the
point of Phase 2. It composes Plan 02 (bsmGreeks), Plan 03 (invertIv), Plan 04
(leg_observations repo + ObservationRow/PendingObs), and Plan 05 (ForReadingRate). The
settlement-aware DTE (D-04) is added here as a pure domain helper. Idempotent re-run is a
SPEC acceptance criterion.
Output: computeBsmGreeks use-case + pending-scan/write repo methods + settlement-aware
computeT, all contract/testcontainers-tested. This plan owns the journal/index.ts
re-export of the Plan 02/03 domain functions.
</objective>
