| Boundary | Description |
|----------|-------------|
| direction tint → verdict | a sign-colored marker must read as long/short, never as a calm/warn/crisis judgment |
| fixture-authored axis → real print | an axis sized from one snapshot must not silently clip a larger real position |
| locked copy → rendered tooltip | CFTC class explanations are authored, not improvised |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-39-02 | Spoofing (false verdict) | COT direction tint | high | mitigate | Marker is only bg-up (net≥0) or bg-down (net<0), never bg-amber/bg-txt; no band segments — a test asserts no COT marker is ever amber, so a sign color can't be misread as a warning verdict |
