<objective>
Scaffold the Bun-workspaces monorepo and make the hexagon dependency law plus strict-TypeScript
both fail the build mechanically. This is the foundation every later plan builds on: the four
packages (`shared`, `contracts`, `core`, `adapters`), the two app shells (`server`, `worker`),
the strict compiler config, the ESLint flat config with boundary enforcement, the root scripts,
and the Vitest workspace harness.

Purpose: FND-01 (workspaces resolve cross-package imports), FND-02 (boundary lint fails the build),
FND-03 (strict TS rejected mechanically), FND-05 (root scripts run across the workspace). Without
this scaffold none of plans 02–06 can run a single test.
Output: a clean checkout where `bun install && bun run typecheck && bun run lint` pass, and where a
deliberately wrong import or an `any` fails the build.
</objective>
