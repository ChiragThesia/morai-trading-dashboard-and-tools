  <name>Shared kernel: Result, assertDefined, OccSymbol</name>
  <files>packages/shared/src/{result,assert,occ-symbol}.ts + colocated *.test.ts + index.ts + vitest.config.ts</files>
  <behavior>
    Result: `ok(v)` → `{ok:true, value:v}`; `err(e)` → `{ok:false, error:e}`; `isOk`/`isErr` narrow.
    assertDefined: throws `Error` containing the message on `null`/`undefined`; otherwise narrows `T|undefined|null → T`.
    OccSymbol: parse a valid OCC symbol (e.g. `SPX   260515C07100000`, root padded to 6, YYMMDD, C/P,
    strike ×1000 zero-padded to 8) into a branded type; format the structured form back; parse→format→parse
    round-trips; invalid input returns a typed failure (Result.err), never throws for bad input.
  </behavior>
  <implementation>Pure TypeScript value objects per RESEARCH Code Examples. OccSymbol logic ported test-first from the trade-advisor CBOE OSI parser (do not copy wholesale — see trade-advisor-inventory.md).</implementation>
</feature>
