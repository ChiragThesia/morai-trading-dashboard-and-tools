## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → GET /api/analytics/macro | The hook sends the bearer/session via apiFetch; a 401 must fail closed (non-retryable), not loop |
| API response → UI | macroResponse.parse validates the body before render (no `as` cast) |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-14-16 | Spoofing | useMacro on an expired session | low | mitigate | 401 → UnauthorizedError, non-retryable (useCot precedent); the app's auth gate handles re-login — no infinite retry storm |
| T-14-17 | Tampering | untyped API body reaching the UI | low | mitigate | macroResponse.parse at the hook boundary — malformed body throws before render, no `as` cast |
| T-14-18 | Denial of Service | missing empty-state → blank/broken card when no data/key | low | mitigate | Explicit empty "run the job to populate" state (never errors, never omitted) — matches the coming-soon-never-breaks discipline |
| T-14-SC | Tampering | npm/bun installs | low | accept | Zero new dependencies (TanStack Query, contracts already present) — no legitimacy gate |
</threat_model>
