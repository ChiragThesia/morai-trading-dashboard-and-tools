<objective>
Extract and build the one part of PayoffChart that no native Recharts primitive covers: the custom
SVG marks. Per RESEARCH D-08 these are exactly three — the KISS edge-arrow glyphs, the short BE-marker
bars, and the EM-band ticks/connector/label. Deliver them as a single, standalone, unit-tested
Customized-layer component so plan 33-06 can wire it into the migrated chart without re-deriving the
clamp/lane logic, and so the phase's most important regression (EM-band page-bleed, commit 2563bd6)
lands as a focused, deterministic unit test.

Purpose: Splitting PayoffChart along this seam keeps both PayoffChart plans within a focused executor
session AND keeps each green: this component is new (no existing file to half-migrate), it needs no
Recharts container to test (it emits raw SVG given a scale), and clamping — forbidden for curves/axes
(use allowDataOverflow there) — is still correct and REQUIRED inside a Customized layer, which axis
clipPath does not cover (RESEARCH anti-pattern note).

Output: `PayoffChartMarks.tsx` (Customized-layer marks) + `PayoffChartMarks.test.tsx` (clamp
regression, edge-arrow lanes, BE markers, absent-safe guards).
</objective>
