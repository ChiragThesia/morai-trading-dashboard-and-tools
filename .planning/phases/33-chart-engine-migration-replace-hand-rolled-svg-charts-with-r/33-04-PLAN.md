<objective>
Migrate the picker's "Term structure + your legs" mini-chart from hand-rolled inline SVG to Recharts.
Its headline idiom is the forward-IV bracket: a partial-width horizontal line between the two leg
x-positions, which is a first-class Recharts ReferenceLine `segment` (D-06 / Pattern 3) replacing a
hand-drawn `M..H..` path string. The guard case (fwdIv null → no bracket, amber "guard" tag, no NaN,
tag stays on-canvas) is a locked behavior (T-18-10 / WR-02) that must survive re-expression.

Purpose: This chart's tests are the most coordinate-specific in the phase (leg dots asserted at exact
xScale/yScale pixels). Recharts owns the scale math, so those assertions re-express to INTENT — dot
color + placement by DTE/IV and ordering — while the load-bearing behaviors (bracket omission guard,
asOf-driven event placement, no-NaN) stay exact.

Output: `TermStructureChart.tsx` on Recharts + re-expressed `TermStructureChart.test.tsx`.
</objective>
