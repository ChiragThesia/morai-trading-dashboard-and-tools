custom marks were already delivered and unit-tested in 33-02, so this plan focuses on the declarative
Recharts body + wiring + test re-expression.

`PayoffChart.test.tsx`. Pure helpers (computeYDomain/buildXTicks/buildXScale/INNER_W) stay unchanged and
re-exported; EDGE_ARROW_LANE_Y is re-exported from PayoffChartMarks so the existing test import resolves.
</objective>
