<objective>
Migrate GexBars (GEX-by-strike bars) off Apache ECharts onto Recharts. This is the one chart moving off
echarts-for-react rather than visx, and it exercises two Recharts idioms the other charts do not: the
horizontal-bar layout (`layout="vertical"`, D-14 / Pitfall 8 naming trap) and per-bar conditional
coloring via `Cell` (D-07). The GEX / OI wall / Volume tab picker is pure shadcn DOM and stays; the
pure helpers (windowStrikes, fmtBn) stay with their existing tests.

Purpose: The current test stubs echarts entirely (canvas getContext is null under jsdom), so the chart
body was never actually asserted. On Recharts the bars render as real SVG under the shared
ResponsiveContainer mock, so this migration also UPGRADES coverage: per-bar sign color becomes testable
instead of stubbed.

Output: `GexBars.tsx` on Recharts + re-expressed `GexBars.test.tsx` (ResponsiveContainer mock replaces
the echarts stub; per-bar Cell colors + reference lines asserted; tab + pure-fn tests preserved).
</objective>
