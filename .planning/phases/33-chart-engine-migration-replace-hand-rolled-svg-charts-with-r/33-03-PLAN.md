<objective>
Migrate GammaProfile (dealer net-gamma profile) from visx to Recharts. It is the smallest curve chart
and a strict subset of PayoffChart's techniques (split-gradient fill + reference lines + numeric axis),
so it is the cheapest place to prove those idioms outside the big chart. It currently has NO test file
(a Wave-0 gap in RESEARCH) — this plan closes that gap with a fresh spec before the swap.

Purpose: The teal/coral split fill is the reusable "value above/below zero" pattern shared with
PayoffChart; proving it here on a small chart de-risks it. The compact/full fixed-size variants
(Analyzer right panel vs Market screen) must keep their exact pixel footprints (D-03).

Output: `GammaProfile.tsx` on Recharts + a new `GammaProfile.test.tsx` (teal/coral split, flip/spot
reference lines, compact vs full sizing, the <2-point null guard).
</objective>
