    - Index 0 (entry baseline): all four cumulatives = 0 when row0 is non-gap; isGap = isGapRow(row0).
    - For each interval [i, i+1] where BOTH rows are non-gap:
        · Δt = (Date(row[i+1].time).getTime() − Date(row[i].time).getTime()) / 86_400_000  (fractional days from `time`, NOT dteFront/dteBack — Pitfall 3)
        · theta bucket = netTheta[i] × Δt   (interval-START greeks — the locked "forward-Euler/sequential" convention, A2)
        · ΔS = spot[i+1] − spot[i];  deltaGamma bucket = netDelta[i]·ΔS + 0.5·netGamma[i]·ΔS²
        · ΔIV = mean(frontIv[i+1], backIv[i+1]) − mean(frontIv[i], backIv[i]);  vega bucket = netVega[i] × ΔIV × 100  (×100 = decimal-IV→vol-points, Pitfall 2; the mean-blend is the locked default per RESEARCH Open Question 1)
        · ΔpnlOpen = pnlOpen[i+1] − pnlOpen[i]  (DOLLARS — never ÷100, Pitfall 1)
        · residual = ΔpnlOpen − theta − vega − deltaGamma  (the EXACT plug — D-05, always kept)
        · cumulatives at i+1 = cumulatives at last non-gap index + this interval's buckets.
    - Gap handling: an interval where row[i] OR row[i+1] is a gap is SKIPPED — no bucket added, the cumulative is NOT advanced across it, and the gap row's own cumulatives are `null` (never zero-filled, never bridged — matches the mockup flush() semantics).
