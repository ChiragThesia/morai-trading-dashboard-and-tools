| payload source/rationale → Tooltip text | Provenance strings rendered as React text nodes (auto-escaped) — no dangerouslySetInnerHTML |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-24-11 | Tampering | Regime response body shape | low | mitigate | regimeResponse.parse in the hook — a bad body throws into the query error state (UI-SPEC error copy), never renders a fabricated chip |
| T-24-12 | Information Disclosure | 401 handling | low | mitigate | Non-retryable UnauthorizedError, cloned from useMacro — no token/response detail surfaced in UI |
| T-24-13 | Spoofing/XSS | source/rationale tooltip text | low | accept | React text-node escaping (no raw HTML injection); provenance strings originate from server-authored domain metadata, not user input |
| T-24-SC | Tampering | npm/pip/cargo installs | low | accept | No new dependency (Tooltip/Badge already installed) — no install/checkpoint applicable |
